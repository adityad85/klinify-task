# Klinify Task
> Uses React and ES6 constructs to build a simple image upload, crop and print.

#What can it do? 

1. Upload image.
2. Crop Image.
3. Preview Image.
4. Print image centered at the top of the page in a new tab with auto closing after print. 
5. clear the input box.
6. click on save to preview the default image.

![](header.png)

## Installation

OS X & Linux:

```sh
npm install or yarn
```

Windows:

```sh
npm install
```

## Development setup

Just clone the repo and run below command to see.

```sh
yarn start
or
npm run start
```
